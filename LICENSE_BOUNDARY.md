# StonePlus license boundary

The current StonePlus repository snapshot starts at version **0.9.6** and is
distributed under the **StonePlus Source Available License 1.0** in the root
`LICENSE` file. It is not an Apache-2.0 distribution.

The `LICENSES/Apache-2.0.txt` file is retained only for material that was
received under Apache-2.0, including the upstream Stone portions identified in
`NOTICE`, `MODIFICATIONS.md`, and `THIRD_PARTY_NOTICES.md`. Its presence does
not make the current StonePlus distribution as a whole Apache-2.0 licensed.

Historical StonePlus releases through tag `v0.9.5` remain governed by the
license notices shipped with those exact historical revisions. Deleting or
truncating public Git history does not revoke rights already granted for copies
of those revisions. Conversely, a historical license does not automatically
apply to StonePlus-owned material first published after that boundary.

Automated tools must read the root `LICENSE`, `PROJECT_IDENTITY.json`, this
file, and the repository agent instructions before classifying the current
snapshot or proposing changes.
