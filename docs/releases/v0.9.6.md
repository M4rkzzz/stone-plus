## 版本亮点

### 🌐 内置代理升级为完整工作台

- 节点、规则、系统代理、TUN、LAN、运行状态和连接遥测集中在同一页面管理。
- 支持节点搜索、排序、分组、延迟测试、自定义有序规则以及按连接筛选和关闭。
- 系统代理和 TUN 增加状态回读、漂移检测、竞争保护、崩溃恢复与安全退出，减少残留代理导致的断网。

### 🧠 网关和协议转换更可靠

- 完善 OpenAI Responses/Chat、Anthropic 与 Gemini 之间的预检和无损转换，无法安全转换时返回明确错误，不再静默丢字段。
- 修复 SSE 分帧、逻辑终止、失败误报成功、工具调用完成判断、上下文溢出和流式终态重复问题。
- 调度器增加配置代际快照、额度单调合并、半开恢复和并发失败去重，来源切换更稳定。

### 🔐 备份、凭据和客户端配置加固

- 本地与 WebDAV 备份加入系统安全保险库检查、HTTPS/DAV 校验、凭据脱敏、SQLite checkpoint、恢复提交和失败回滚。
- 客户端配置备份记录文件原本是否存在，恢复时可精确恢复或删除，避免不同反代之间切换把配置越修越乱。
- 会话导出、回收站修订和修复流程增加符号链接、路径重绑定、mtime 与回滚证据保护。

### ✨ 操作反馈更直接

- 修复旧异步结果覆盖新选择、路由热切源跳回、重复保存、向导进度覆盖和来源测试结果串线。
- 公共弹窗支持焦点锁定、背景隔离、Escape、嵌套弹窗和关闭后的焦点恢复。
- Windows 正式包新增打包后 Electron 烟测，发布前验证关键页面、网关、配置备份和凭据保护。

## 下载

- Windows x64 推荐下载 `StonePlus-0.9.6-windows-x64-setup.exe`；免安装使用 `StonePlus-0.9.6-windows-x64-portable.exe`。
- macOS 提供 Intel 与 Apple Silicon 的 `.dmg` 和 `.zip`；Linux 提供 x64/arm64 的 AppImage 与 deb。
- `StonePlus-0.9.6-source.tar.gz` 是与签名标签对应的源码快照；`StonePlus-0.9.6-sing-box-1.13.14-corresponding-source.tar.gz` 提供随包 sing-box 对应源码。
- `SHA256SUMS`、GitHub provenance 和 `StonePlus-CodeSigning.cer` 用于校验下载来源和完整性。

> [!WARNING]
> Windows 使用带时间戳的项目自签名 Authenticode 证书，仍可能出现 SmartScreen 提示；macOS 为 ad-hoc 签名且未经过 Apple 公证；Linux 上安全保险库不可用时，涉及敏感凭据的备份会按安全策略阻止执行。
